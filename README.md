## Features

- 🐱 **Cat animation** on startup
- 📊 **Active connections** monitoring (TCP/UDP)
- 📈 **Bandwidth tracking** with sparkline graphs
- 📦 **Packet capture** with protocol parsing (requires sudo)
- 🔍 **Process mapping** to see which apps use the network
- 💾 **Export data** to JSON/CSV
- ⚙️ **Configurable** via `~/.config/mewn/config.toml`

## Installation

### From source (development)

```bash
git clone https://github.com/yourusername/mewn.git
cd mewn
cargo build --release
```

### Install globally

```bash
# Install via cargo (puts binary in ~/.cargo/bin/)
cargo install --path .

# Or copy binary to PATH
cp target/release/mewn /usr/local/bin/

# Verify installation
mewn --version
```

After installation, the binary `mewn` will be available in your PATH. Type `mewn` to launch the dashboard.

## Usage

```bash
# Run the dashboard (default)
mewn

# Run with packet capture (requires root)
sudo mewn

# Export data
mewn export --format json --output data.json
mewn export --format csv --output data.csv

# Show version
mewn version

# Show help
mewn --help
```

### What Happens When You Run `mewn`

1. Terminal enters alternate screen
2. Cute Braille cat animation plays (2 seconds)
3. Press any key to skip animation
4. Dashboard appears with 4 tabs:
   - **Connections** — Live TCP/UDP connections
   - **Bandwidth** — Per-interface stats with graphs
   - **Packets** — Live packet capture (requires sudo)
   - **Processes** — Per-process network usage
5. Press `q` to quit and return to terminal

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `q` | Quit |
| `Tab` | Switch tabs |
| `?` | Show help |
| `t` | Filter: TCP only |
| `u` | Filter: UDP only |
| `i` | Filter: ICMP only |
| `d` | Filter: DNS only |
| `a` | Filter: All protocols |
| `n` | Sort processes by name |
| `p` | Sort processes by PID |
| `c` | Sort processes by connections |

## Requirements

- macOS (primary platform)
- Rust 1.70+ (for building)
- Root/sudo (for packet capture)

## Configuration

Edit `~/.config/mewn/config.toml`:

```toml
interface = "en0"
refresh_rate_ms = 1000

[theme]
tcp_color = "green"
udp_color = "blue"
dns_color = "yellow"
```
